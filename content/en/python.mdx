The Python SDK is async-first, using `aiohttp` + `asyncio`. Decorator-style message handlers, dataclass types, 17 tests. Requires Python ≥ 3.9.

<Section id="install" title="Install">

```bash
pip install wechatbot-sdk
```

</Section>

<Section id="quick-start" title="Quick start">

```python
from wechatbot import WeChatBot

bot = WeChatBot()

@bot.on_message
async def handle(msg):
    await bot.send_typing(msg.user_id)
    await bot.reply(msg, f"Echo: {msg.text}")

bot.run()  # login + start in one call
```

Or with full async control:

```python
import asyncio
from wechatbot import WeChatBot

async def main():
    bot = WeChatBot()
    await bot.login()

    @bot.on_message
    async def handle(msg):
        await bot.reply(msg, f"Echo: {msg.text}")

    await bot.start()

asyncio.run(main())
```

<Section id="messages" title="Message handling">

```python
@bot.on_message
async def handle(msg):
    print(f"[{msg.type}] {msg.user_id}: {msg.text}")

    for img in msg.images:
        print(f"  Image: {img.url}")

    for voice in msg.voices:
        print(f"  Voice: {voice.text} ({voice.duration_ms}ms)")

    for file in msg.files:
        print(f"  File: {file.file_name} ({file.size} bytes)")

    if msg.quoted_message:
        print(f"  Quoted: {msg.quoted_message.title}")
```

</Section>

<Section id="sending" title="Sending">

```python
# Reply (auto context_token + stop typing)
await bot.reply(msg, "Hello!")

# Send to user (needs prior context_token)
await bot.send(user_id, "Proactive message")

# Typing indicator
await bot.send_typing(user_id)
await bot.stop_typing(user_id)
```

</Section>

<Section id="crypto" title="AES-128-ECB crypto">

```python
from wechatbot import (
    generate_aes_key, encrypt_aes_ecb, decrypt_aes_ecb, decode_aes_key
)

key = generate_aes_key()
ct = encrypt_aes_ecb(b"Hello", key)
pt = decrypt_aes_ecb(ct, key)

# Decode protocol key (all 3 formats)
k = decode_aes_key("ABEiM0RVZneImaq7zN3u/w==")       # base64(raw)
k = decode_aes_key("00112233445566778899aabbccddeeff") # hex
```

</Section>

<Section id="types" title="Types (dataclasses)">

```python
@dataclass
class IncomingMessage:
    user_id: str
    text: str
    type: Literal["text", "image", "voice", "file", "video"]
    timestamp: datetime
    images: list[ImageContent]
    voices: list[VoiceContent]
    files: list[FileContent]
    videos: list[VideoContent]
    quoted_message: QuotedMessage | None
    raw: dict
```

Error hierarchy: `WeChatBotError` → `ApiError`, `AuthError`, `NoContextError`, `MediaError`
